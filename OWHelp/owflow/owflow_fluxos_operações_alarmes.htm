Alarmes de um Fluxo

 

O botão Alarmes permite configurar os alarmes do fluxo à semelhança da configuração dos alarmes de uma etapa. 

Para adicionar um alarme a um fluxo, seleccionar o Fluxo e pressionar o botão “Alarmes”.

 

No écran que aparece de seguida, premir o botão “Inserir”, preencher os dados e seguidamente confirmar.

Para modificar um alarme de um fluxo, pressionar o botão “Alarmes”, seleccionar o alarme que pretende modificar e

premir o botão “Modificar”. Depois disso, preencher os dados que pretende modificar e seguidamente confirmar.
