Inserir Fluxo

Esta opção permite criar um novo fluxo. Apenas é necessário introduzir o código e o nome e depois premir o botão “Inserir”. Quando é criado um novo fluxo, este é inicialmente classificado como estando "em construção".

Não podem existir fluxos com o mesmo código ou com o mesmo nome. Depois de publicado, o fluxo passa a estar “em Produção” .

Dentro do Fluxo é possível ainda:

A) Inserir Etapas

Seleccionar a pasta “Etapa” e premir o botão “Inserir”. Este procedimento vai permitir adicionar uma etapa de um fluxo. 

Nota: Para se poder inserir uma etapa, é necessário “Inserir” previamente o fluxo.

O botão de pesquisa do campo “Executante” acede à janela de pesquisa de utilizadores e/ou grupos e permite inserir o utilizador ou grupo de utilizadores que vai ter acesso à etapa.   

Depois de preencher o Nome da Etapa e o Executante (Entidade ou Grupo), premir o botão “Confirmar” . O écran de listagem de etapas do fluxo aparece novamente .

 

B) Modificar Etapa

Altera os dados da etapa seleccionada. Para modificar uma etapa, seleccionar a etapa pretendida e premir o botão “Modificar”.  Proceder às alterações e premir o botão “Confirmar”. 

C) Apagar Etapa

Remove do fluxo a etapa seleccionada. Para apagar uma etapa, seleccionar a etapa e premir o botão “Apagar” na figura inicial. Aparece uma janela de confirmação. Premir o botão “Sim”, se desejar de facto apagar a etapa. A etapa desaparece da lista.

 

D)Alterar a ordem das Etapas

 

Subir – Passa a etapa seleccionada para cima da etapa anterior.

Descer – Passa a etapa seleccionada para a baixo da etapa seguinte.

E) Definir Alarmes das Etapas

Permite visualizar todos os alarmes da etapa seleccionada.  Para inserir um alarme numa etapa, seleccionar a etapa pretendida e premir o botão “Alarmes”.  Aparece um écran semelhante ao da seguinte figura:

 

Para inserir um alarme novo seleccionar o botão “Inserir”,  preencher os dados pedidos e confirmar.

Para modificar um alarme já existente, seleccionar a etapa, premir “Alarmes”. Seleccionar o alarme a alterar e premir o botão “Modificar” . Proceder às alterações pretendidas e premir o botão “Confirmar”.

 

F) Definir Acessos

Permite configurar os utilizadores e grupos com acesso de consulta aos processos criados com base no fluxo.
