Chaves do Registo

Durante a instalação do OfficeWorks são registadas algumas chaves de configuração no registry. Estas chaves não devem ser alteradas/Removidas sem a assistência de um Técnico especializado da Magnetik.
As chaves são registadas na pasta

HKEY_LOCAL_MACHINE\SOFTWARE\OW
e são descritas de seguida:

 BDNome do Servidor de Base de Dados SQL
 ExchangeServer Nome do Servidor de Exchange
 FileServerNome do Servidor de Ficheiros
 IndexServerNome do Servidor de Indexação *
 ServerNome do Servidor de OfficeWorks
 WebserverServidor WEB (IIS) onde se encontra o OfficeWorks
 VirtualDirectoryDirectoria Virtual do IIS que aponta para o OfficeWorks
 WebserviceNome do Servidor onde se encontra o Webservice
 TraceUtilizada pelo Webservice (0 por defeito)
 Local_Temp_Dir Directoria Temporária
 EventViewerChave de serviço
 AppChave de serviço

* Utilizado apenas no módulo Full Text Search
