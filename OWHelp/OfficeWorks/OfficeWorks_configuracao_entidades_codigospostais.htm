Códigos Postais

No módulo de Códigos Postais é permitido ao gestor introduzir novos códigos postais, à medida que vão sendo necessários.

Para introduzir o Código Postal novo, o Gestor deverá preencher o Código e a Descrição correctamente e premir o botão .

Pode repetir o Código e a Descrição, mas nunca os dois juntos. Ou seja, o Código “2780-278” poderá ter outra descrição de Código Postal e a Descrição "Oeiras” poder ter outro código de Códigos Postais.

É também através deste écran que pode modificar ou apagar definitivamente um Código Postal já existente. Para isso, basta seleccionar o Código Postal que se pretende modificar, proceder à modificação desejada, depois de pressionar o botão .

Para apagar, basta seleccionar o Código Postal e premir . 

Só pode ser apagado um Código Postal que não esteja associado a uma Entidade.
