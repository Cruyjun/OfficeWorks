Enviar uma mensagem de Workflow

O Correio OfficeWorks.Net dispõe de mensagens do tipo WorkFlow. Uma mensagem deste tipo é enviada para o primeiro utilizador da lista de destinatários, circulando depois para o segundo destinatário, deste para o terceiro e assim sucessivamente.

A cada nome da lista de destinatários corresponde uma etapa da circular, que terá de ser completada para que a mensagem seja enviada para o destinatário seguinte.

Cada um dos destinatários que recebe a mensagem deverá lê-la, podendo adicionar mais texto, bem como associar os documentos que entender. Finalmente, deverá Despachar a circular, para esta passar à etapa seguinte.

Por outro lado, se tiver algo negativo a dizer sobre o seu conteúdo, poderá interromper o percurso da circular, Indeferindo a mensagem.
Cada circular é identificada através de um número único, atribuído pelo sistema no momento em que é enviada pelo primeiro remetente.

Envio

Para enviar uma mensagem de correio tipo WorkFlow deverá seleccionar a opção Escolher Formulário… do menu Novo.



No écran que surge deverá seleccionar o formulário OfficeWorks.



Após esta operação surge o écran de Envio de Workflow, apresentado na figura seguinte



Para enviar um correio por WorkFlow o utilizador deverá compor a mensagem de acordo com o seguinte:

No campo Para, o utilizador deverá especificar o(s) destinatário(s) da mensagem. Se pretender enviar para mais do que um destinatário, deverá separar os seus nomes com ponto-e-vírgula, ou então, carregar no botão Para:, este permite aceder a uma caixa de diálogo semelhante à da figura , onde poderá seleccionar os destinatários.

No campo Assunto, o utilizador deverá digitar o assunto da mensagem.



No campo Cc (Com conhecimento) devem incluir-se os utilizadores para os quais pretendemos enviar uma cópia da mensagem. No caso de se pretender enviar uma cópia "secreta" (desconhecida dos restantes utilizadores) para determinados utilizadores, deveremos inseri-los no campo Sc (Sem conhecimento). Os botões “Cc…” e “Sc…”, permitem aceder a um écran semelhante ao da figura.

O texto da mensagem e os documentos que pretender enviar, deverão ser colocados no campo Mensagem, que se encontra por baixo do campo Assunto.

Para associar documentos à mensagem deverá seleccioná-los no Explorer do Windows e arrastá-los para cima da janela aberta do Outlook. Os ícones correspondentes aos documentos surgirão no campo Mensagem. Uma outra possibilidade é carregar no botão (para anexar ficheiros) da barra de ferramentas:


Nota: Se o PC possuir equipamento para gravação poderá criar um ficheiro de som e associá-lo tal como foi explicado em relação aos documentos.  

Se o utilizador pretender apagar um determinado documento, deverá seleccioná-lo e pressionar a tecla Delete.

Para mais informações acerca das várias opções de envio de correio, poderá consultar o Help Online do Outlook.

É importante referir, no entanto, que a ordem pela qual a mensagem circulará entre os utilizadores é aquela pela qual os seus nomes surgem no campo Para.

Depois de compor a mensagem deverá pressionar o botão “Enviar”, que se encontra na Barra de Ferramentas do Outlook, para dar início à primeira etapa do percurso do WorkFlow.
