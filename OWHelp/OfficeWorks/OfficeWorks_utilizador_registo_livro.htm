Inserir Registo

O botão à direita do campo Livro, no écran da figura, permite aceder à lista de livros disponíveis, onde deve seleccionar o que pretende, o que provoca o preenchimento do campo Livro.



Se o utilizador não tiver permissão para inserir registos, surgirá uma mensagem informando-o desse facto. Caso contrário, deverá seleccionar o livro, da lista de livros existentes e a que tem acesso, em que pretende inserir o registo. Depois de seleccionar o livro pretendido, serão apresentados os campos a preencher, semelhantes aos da figura.



Este diálogo permite efectuar o registo de toda a correspondência que se encontra dividida em diferentes campos. À excepção dos campos Livro, Ano/Número e Data do Registo, os campos são dinâmicos, ou seja, cada livro é configurado pelo Gestor apenas com os campos que fazem sentido. A qualquer momento é possível retirar (se ainda não tiver sido utilizado para nenhum registo) ou introduzir um novo campo.

 Depois de preencher todos os campos necessários, o utilizador deverá pressionar o botão Inserir. Se pretender limpar os campos, deverá utilizar o botão Limpar. Para manter os campos preenchidos de um registo para o outro com os mesmos valores, deverá preencher o quadrado que se encontra à esquerda do campo Entidade. O * (à direita) significa que o campo é obrigatório, e que não conseguirá inserir o registo com o campo em Lopes.   

É o gestor do serviço de Registo de Correspondência que define, para cada livro, quais os campos de preenchimento obrigatório. Assim, no écran de inserção de registos, os campos cujo preenchimento é obrigatório poderão diferir consoante o livro seleccionado. Fica aqui a descrição dos campos mais relevantes:

* Livro : código e nome do livro onde se pretende inserir o registo.
* Ano/Número : ano e número de registo, que constituem (juntamente com o livro) a identificação da ficha de registo. Se a opção Numeração Automática estiver activa, este campo será automaticamente preenchido pelo sistema. Porém, em modo de Numeração Manual , será o utilizador a introduzir estes dados. A numeração não poderá, contudo, ser igual a nenhuma já associada a registos anteriores nesse livro.
* Data de Registo : data de inserção da ficha de registo. Este campo é automaticamente preenchido com a data actual. Poderá, no entanto, ser alterado pelo utilizador, com o auxílio de um calendário.
* Tipo Documento :código e descrição do tipo de documento a registar (carta, ofício, etc.).
* Referência : referência (ou identificação) do documento associado à ficha de registo.
* Data : data do documento. Este campo pode ser preenchido com o auxílio do calendário.
* Entidade : descrição da entidade associada a este registo.
* Classificação : código para os 5 níveis da classificação a associar ao registo.
* Assunto : assunto do documento.
* Observações : observações relativas ao documento.
* Palavra Chave : palavras-chave associadas ao registo.


Para preencher os campos Livro, Documento, Entidade, Classificação e Palavras-Chave , o utilizador deve seleccionar um dos elementos que constam das respectivas tabelas. O acesso a estas tabelas faz-se através do botão ou , colocado à direita de cada um destes campos.

Nota : se já tiver preenchido alguns campos, e alterar o Livro seleccionado, o conteúdo inserido nos campos que dele dependem serão apagados.
