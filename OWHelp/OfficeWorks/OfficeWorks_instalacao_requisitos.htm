Requisitos de Hardware e Software

Requisitos de Administração para Configuração da Aplicação
Conhecimentos necessários, a nível de administração, das seguintes plataformas:
1. SQL Server 2000
2. Internet Information Services 5.0/6.0
3. Exchange Server 2000

Requisitos de Sistema Operativo

Os sistemas operativos base para a instalação do Servidor OfficeWorks são o Microsoft Windows 2000 Server com o Service Pack 4 ou o Microsoft Windows 2003 Server.

Caso o SO seja o Windows 2000/2003, o mesmo terá que ser instalado sobre um file system NTFS.

O login de sistema operativo que efectua a instalação tem que ter privilégios de administrador.

O Correio Circular está testado e funcional nos sistemas operativos Microsoft Windows 98, Microsoft Windows NT 4.0 Server com o Service Pack 6a ou o Microsoft Windows 2000 Server/WorkStation.

Requisitos de Hardware

Para normal funcionamento do OW.Net será necessário que as seguintes plataformas estejam devidamente instaladas e configuradas.

Servidor
1. SQL Server 2000 SP3
2. Internet Information Services 5.0 ou 6.0 com ASP a funcionar correctamente
3. Exchange 2000 SP3 ou 2003 [Correio Electrónico]
4. Framework .Net 1.1 com ASP.NET a funcionar correctamente

Cliente
1. Internet Explorer 6.0 SP1 ou compatível
2. CDO 1.2x [Correio Circular]
3. Outlook 2000 SR1/XP/2003 [Correio Circular]
