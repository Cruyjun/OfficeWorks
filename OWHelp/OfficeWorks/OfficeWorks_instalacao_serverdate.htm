Data dos Servidores

Caso os servidores (base de dados, web, etc. ) se encontrem em máquinas diferentes é necessário que as datas dos mesmos estejam sincronizadas de forma a garantir coerência nos vários componentes de um item, gerados nas respectivas plataformas (e.g. registo na base de dados vs HTML dinâmico gerado).
