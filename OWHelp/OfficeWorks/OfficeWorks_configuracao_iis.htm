Configuração do Internet Information Services (IIS)

O Setup do OfficeWorks faz a configuração automática do Internet Information Services (IIS). Se o utilizador que faz a instalação o pretender fazer manualmente, tem de configurá-lo do seguinte modo:
1. Abrir o Internet Information Services (IIS)

Menu Start»Programs»Administrative Tools»Internet Information Services(IIS) Manager

2. Criar uma Directoria Virtual com o nome "OfficeWorks"
