Perfis de Impressão / Exportação

Para se poder imprimir o resultado de uma pesquisa, ou exportar os dados para um ficheiro, é necessário proceder a uma pesquisa, para obter a listagem pretendida.

A figura seguinte mostra o resultado de uma determinada pesquisa.


Ao obter este resultado, proveniente de uma pesquisa, podemos observar dois botões, um de "Imprimir", que serve para se obter uma impressão dos registos, e um de "Exportar" , que permite exportar os registos (apenas com os campos escolhidos na configuração), para um ficheiro de Excel para posterior tratamento, divulgação ou observação dos mesmos.

Se o utilizador tiver permissões, estará também visível o botão de passagem de registos para histórico "Histórico".

Independentemente do perfil, se não se pretender imprimir / exportar todos os registos retornados na pesquisa, é necessário fazer um “check” no local que antecede as iniciais do livro onde está o registo.

Perfil de Impressão

Posteriormente à escolha dos registo a imprimir, pressiona-se o botão de "Imprimir". Aparece então o seguinte écran:


Se não tiverem sido escolhidos registos, o programa irá imprimir todos os registos retornados. Na parte inferior da janela, é permitido escolher um mapa de impressão previamente configurado no menu “Configuração”, ou um mapa detalhado / detalhado simples já existente, com determinados campos seleccionados, consoante o objectivo das listagens.

O perfil de Impressão através de um Mapa pré definido, e apenas pelo Mapa, permite ao utilizador seleccionar o número de linhas que pretende entre cada registo, para facilitar a visualização no papel

Depois de tudo confirmado, premir o botão "Imprimir" e aparece a janela com um “Preview” da impressão pretendida:


O segundo botão serve para definir as margens, cabeçalhos, tipo de papel e orientação da folha (se vai imprimir na vertical ou horizontal).

Depois de confirmar que a impressão corresponde às expectativas e tem os dados correctos, carregar no botão “Print” ou “Imprimir” que aparece no canto superior esquerdo.

Ao fazer isto, aparece o écran de escolha das impressoras:


Após a escolha da impressora, carregar no botão “Print” ou “Imprimir” e obtém-se o documento pretendido.

Perfil de Exportação

Posteriormente à escolha dos registos a exportar para ficheiro, pressiona-se o botão "Exportar" . Aparece então o seguinte écran:


Se não tiverem sido escolhidos registos, o programa irá exportar para o ficheiro de Excel, todos os registos retornados na pesquisa.

Na parte inferior da janela, é permitido escolher um perfil de exportação previamente configurado no menu “Configuração” com os campos necessários a cada departamento/utilizador. Depois de tudo confirmado, premir o botão "Confirmar". Nessa momento é criado um ficheiro com os registos exportados.
