Configuração de Utilizadores

No serviço de Configuração de Utilizadores, vão ser criadas as condições necessárias para que os utilizadores da aplicação OfficeWorks dentro da instituição, possam aceder ao sistema.

Esta secção contém os seguintes tópicos:

 Tópico  Descrição
Criar um novo utilizador Descreve o procedimento para a criação de um novo utilizador de OfficeWorks
Modificar um utilizador Descreve o procedimento para a modificação dos dados de um utilizador de OfficeWorks
Remover um utilizador Descreve o procedimento para a Remoção dos dados de um utilizador de OfficeWorks
Activar/Desactivar um utilizador Descreve o procedimento para a Remoção dos dados de um utilizador de OfficeWorks
Imprimir Utilizadores Descreve o procedimento para a imprimir uma lista com os utilizadores de OfficeWorks
Acessos dos Utilizadores Descreve o procedimento para dar acessos aos utilizadores de OfficeWorks
