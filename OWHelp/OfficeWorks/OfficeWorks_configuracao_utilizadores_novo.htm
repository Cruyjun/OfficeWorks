Novo Utilizador

Para criar um novo utilizador, o Gestor deverá seleccionar o menu Configuração no écran principal, que irá retornar o écran para criar o utilizador.

Deverão então ser preenchidos os dados do utilizador, conforme estejam no Outlook, para o Correio Circular funcionar correctamente.


 

Depois de confirmar os dados do utilizador e os perfis de acesso aos respectivos módulos, deverá então seleccionar "Novo" no écran de Novo Utilizador.   Neste momento, o campo do Domínio permite escrever até 449 caracteres e o campo do Utilizador até 450 caracteres.
