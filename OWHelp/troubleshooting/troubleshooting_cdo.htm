CheckList de Instalação do CDO

[ ] Ping para o servidor EXCHANGE funciona

[ ] O CDO 1.2x está instalado a partir do setup do Office 2000 SR1

[ ] Só existe um ficheiro CDO.DLL e este encontra-se em

C:\Program Files\Common Files \System\Mapi ou

C:\Program Files\Common Files\System\Mapi\1033 ou C:\Program Files\Common Files\System\Mapi\2070

[ ] O utilizador tem uma mailbox definida no EXCHANGE

[ ] O utilizador tem permissões suficientes para aceder à mailbox

[ ] A directoria virtual no IIS só tem o modo de segurança Basic Autentication

[ ] O servidor EXCHANGE está devidamente configurado no registry onde está instalado o

[ ] OW.Net (chave HKEY_LOCAL_MACHINE/Software/OW/ExchangeServer)

[ ] O valor do registry HKEY_LOCAL_MACHINE/Software/Microsoft/Windows Messaging

[ ] Subsystem\ProfileDirectory não está devidamente configurado.

.A pasta referida em ProfileDirectory do registry não tem permissões Full Control para todos os Utilizadores.
