CheckList de Instalação do OfficeWorks

[ ] Instalar framework.net 1.1

[ ] Copiar counteudo de wwwRoot (webctrl_client) para inetpub\wwwRoot do IIS

[ ] Criar a base de dados OW no enterprise manager

[ ] Criar o login OW com a password nicedoc

[ ] Configurar permissões public e db_owner para a base de dados OW

[ ] Correr o script sql, no query analyzer, para criar a base de dados (Registo\SQL)

[ ] Executar o setup e instalar (Registo)

[ ] Verificar se directorias virtuais do IIS têm basic autentication

[ ] Adicionar os ficheiros da pasta GAC para a GAC do .Net

[ ] Verificar o procedimento para configurar o Servidor de ficheiros atraves de drive de rede mapeado (em anexo) ou http://support.microsoft.com/default.aspx?scid=kb;en-us;307901

[ ] Verificar o procedimento se o IIS estiver num controlador de domínio (Server Application Unavailable) (em anexo) ou http://support.microsoft.com/default.aspx?scid=kb;en-us;315158
