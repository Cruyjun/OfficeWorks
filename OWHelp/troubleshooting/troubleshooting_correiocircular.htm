CheckList de Instalação do Correio Circular

[ ] O OutLook não pode estar instalado na mesma máquina onde está o Exchange

[ ] O utilizador está activo no OfficeWorks

[ ] A delivery está a ser na MailBox e não num personal folder

[ ] O utilizador tem a descrição no OfficeWorks igual à da caixa de correio

[ ] O endereço de mail configurado no OfficeWorks está correcto

[ ] O CDO está instalado

[ ] O utilizador tem permissões para executar o CDO

[ ] O utilizador tem permissões para executar o código de formulários organizacionais

[ ] Em ambiente de Terminal Services o Outlook 2002/2003 não instala a DLL OUTLVBS.DLL em Programas\Microsoft Office\Office10

[ ] O site do OfficeWorks está nos trusted sites

[ ] O custom level dos trusted sites está devidamente configurado:

Initialize and script ActiveX Controls not marked as safe

Run ActiveX controls and plug-ins

Script ActiveX controls marked safe for scripting

Scripting
