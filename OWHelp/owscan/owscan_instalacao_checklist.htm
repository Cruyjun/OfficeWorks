Check List - Para que o OWScan funcione correctamente, há que ter em conta a execução dos seguintes pontos:

Instalar o Webservice

Fazer ping ao Webservice ou Correr o webservice no IE

Abrir o Browser e correr o OfficeWorks onde vai ser feito o registo

Correr o Setup do OWScan

O servidor EXCHANGE está devidamente configurado no registry onde está instalado o OW.Net (chave HKEY_LOCAL_MACHINE/Software/OW/ExchangeServer)

A directoria virtual do servidor WEB está devidamente configurada no registry (chave HKEY_LOCAL_MACHINE/Software/OW/VirtualDirectory)

O WEBSERVICE está devidamente configurado no registry (chave HKEY_LOCAL_MACHINE/Software/OW/WebService)

A Directoria Local Temporaria está devidamente configurada no registry (chave HKEY_LOCAL_MACHINE/Software/OW/LOCAL_TEMP_DIR)
