Requisitos de Software

O sistema operativo base para a instalação dos componentes para os templates é a plataforma cliente Microsoft (Windows 2000 professional, Windows xp).

Para normal funcionamento do OWScan será necessário que os seguintes componentes estejam devidamente instalados e configurados:

1. WebService do OfficeWorks (OfficeWorks API) correctamente instalado e configurado na mesma máquina ou acessível em uma máquina da rede com ligação ao servidor OfficeWorks

2. Software de digitalização do Scanner (com Driver TWAIN) devidamente instalado e configurado
