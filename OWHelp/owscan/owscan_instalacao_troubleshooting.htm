Resolução de Problemas

1. Method '~'of Object '~' Failed

Ao correr o OWScan.exe, se der o Erro



Verificar se o webservice está bem configurado no registry
(chave HKEY_LOCAL_MACHINE/Software/OW/WebService)



2. Erro ao digitalizar o documento

Em alguns scanners foi reportado que, ao fazer a primeira digitalização, já com o OWScan a funcionar, aparece um erro. Tentar novamente. Se o problema persistir, contactar os serviços de suporte a clientes da Magnetik.

3. Erro: “To help protect your security, Internet Explorer has restricted …”

Ao iniciar o registo, o OwScan vai tentar abrir uma página no Browser. Se ao abrir essa página aparecer uma barra amarela com o seguinte texto: ”To help protect your security, Internet Explorer has restricted this file from showing active content that could access this computer. Click here for options...” , estamos perante uma medida de segurança implementada com o Service Pack 2 do Windows XP.



Para contornar esta medida de segurança, tentar o método abaixo indicado

1) clicar na Mensagem a amarelo e seleccionar “Allow Blockec Content”. Aparecerá a seguinte mensagem:



2) seleccionar “Yes”. Isto resolverá o problema no momento. Para resolver de vez sem mais avisos deverá alterar os Security Settings do Internet Explorer da seguinte maneira:

1. Abrir menu Tools » Internet Options » Advanced Tab
2. Scroll até Security Section
3. Ligar a opção “Allow active content from CDs to run on my computer”
4. Ligar a opção “Allow active content to run in files on My computer”
